Google Fonts 內一些被隱蔽的字體


Google Fonts 是一個很好的資源，其給網站使用的 API 更是被此網所用。但是，有三個字體是被隱蔽。

根據 Wayback Machine，在二零一六年時，有五個繁體中文字體曾在 Early Access 內試用過。它們順序為：cwTeXFangSong（仿宋）、 cwTeXHei（黑體）、 cwTeXKai（楷體）、 cwTeXMing（明體）、 及 cwTeXYen（圓體）。其中，cwTeXHei（黑體）和 cwTeXMing（明體）已被思源黑體與思源宋體取代。 但其他就在2017年左右莫名其妙地消失。現今，這些字體只能在 Wayback Machine 或 Google Fonts 內的 WebFonts API 使用。


以下就為這些字體的樣本：

楷體：
南去經三國，東來過五湖
圓體：
南去經三國，東來過五湖
仿宋體：
南去經三國，東來過五湖

如以上的字體未能加載，請按下此按鈕去觀看 png 版。
